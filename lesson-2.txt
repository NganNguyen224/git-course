1. Giải thích câu lệnh
git init: Khởi tạo thư mục được quản lý bởi Git.
git add <file_name>: Thêm 1 file vào vùng staging.
git add . : Thêm toàn bộ files vào vùng staging.
git commit -m "message": Commit code vào vùng Staging Area
