1. Giải thích câu lệnh
git commit --amend: dùng để sửa đổi commit gần nhất mà không tạo một commit mới.
git commit --amend -m "message": thay đổi nội dung message đã commit sai.
git restore --staged <file>: restore 1 file cụ thể từ vùng Staging về vùng Working Directory.
git reset HEAD~1: restore 1 commit gần nhất từ vùng File repository về vùng Working Directory.
